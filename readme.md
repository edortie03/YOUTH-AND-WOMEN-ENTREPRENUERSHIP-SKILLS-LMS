# 📚 Youth & Women Entrepreneurship LMS (PWA)

A mobile-first **Learning Management System (LMS)** designed to deliver practical entrepreneurship training to youth and women. The platform is optimized for **low-bandwidth environments** and works as a **Progressive Web App (PWA)**, allowing users to learn anytime, anywhere.

---

## 🎯 Project Objective

To eliminate barriers to entrepreneurship education such as:

* Transport costs
* Limited training spaces
* Time constraints

By providing a **digital learning platform** accessible via smartphones and web browsers.

---

## 👥 Target Users

* **Learners:** Youth and women entrepreneurs
* **Instructors/Admins:** Training organizations and educators

---

## ⚙️ Tech Stack

### Backend

* Django
* Django REST Framework (DRF)
* JWT Authentication

### Database

* PostgreSQL (Production)
* SQLite (Development/Demo)

### Frontend

* HTML
* CSS
* Bootstrap
* JavaScript

### Media Handling

* Cloudinary (video hosting & optimization)

### PWA Features

* Web App Manifest (`manifest.json`)
* Service Worker (`service-worker.js`)
* Offline support (non-video content)

---

## 🚀 Features

### 👤 Learner Features

* Register & login
* Browse and enroll in courses
* Watch video lessons (Kiswahili-focused)
* Take quizzes
* Participate in discussion forums
* Track progress
* Earn certificates

---

### 🛠️ Admin Features

* Create and manage courses
* Upload videos and learning materials
* Create and manage quizzes
* Monitor student progress
* View performance analytics and reports
